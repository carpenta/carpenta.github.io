## R 로 데이터 분석 해보려고..

### 우선 plot 툴 부터 깔고..
```
> install.packages("ggplot2")
> library(ggplot2)
```

### R 좀 쉽게 쓰자고 dplyr 깔고..
```
> install.packages(c("dplyr", "hflights"))
> library(dplyr)
> library(hflights)
```
